GUNNA E-Commerce

A fashion store website built with HTML and CSS.

About
------

GUNNA is a front-end project where I created a multi-page e-commerce style website.

The goal was to practice building a real-looking website layout, working with responsive design, and organizing different pages.

Pages
-----

• Home page  
• Product collection pages  
• About page  
• Contact page  

Features
--------

• Responsive layout for different screen sizes  
• Navigation between pages  
• Product cards and collections  
• Contact form design  
• Shopping cart button UI  
• Hover effects and custom styling  

Built With
----------

• HTML5  
• CSS3
